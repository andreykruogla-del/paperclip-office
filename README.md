# Paperclip Office

This project is not affiliated with the official Paperclip project.

Paperclip Office is an independent open-source visual control room built for Paperclip-based systems.

## Goal

Paperclip Office helps developers and teams see:

- which agents are active
- what tasks they are working on
- how tasks are handed off between agents
- where failures happen
- how much runs cost

## Status

Early MVP.
